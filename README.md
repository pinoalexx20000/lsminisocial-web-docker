# LsMiniSocial - Web Application with Docker

Academic web application developed as part of the Web Programming subject at La Salle Campus Barcelona.

The project is a small social web application built with a containerized development environment using Docker. It includes separate services for the web server, backend application, database and phpMyAdmin.

## Features

- Web application structure
- PHP/CodeIgniter backend
- MySQL database connection
- Docker-based development environment
- Nginx web server
- phpMyAdmin for database management
- Basic user and data management features

## Technologies

- PHP
- CodeIgniter
- MySQL
- Docker
- Nginx
- phpMyAdmin
- HTML
- CSS
- Git/GitHub

## What I learned

This project helped me understand how to configure a web development environment with Docker, connect backend services with a database and work with a more realistic web application architecture.

It also allowed me to practice backend development, database management, environment configuration and web application structure.
